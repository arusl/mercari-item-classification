Welcome!

This application is a dummy project made during Build @ Mercari 2021 to learn about the Deployment of ML Models.
This application currently has one API serving a model for classifying a product category based on its name.
There are 10 main categories that a product can be classified into, which are:
  1. Beauty
  2. Handmade
  3. Vintage & Collectibles
  4. Men
  5. Women
  6. Kids
  7. Electronics
  8. Sports & Outdoors
  9. Home
  10. Other
You can try to use the model too by sending a POST request to https://udemy-mercari-api.herokuapp.com/api/v1/predict,
and send a JSON body with key="item_name" and value="the name of the item you want to predict".

Example request JSON: { "item_name": "logitech mouse" }
Example response JSON: { "item_name": "logitech mouse", "label": "Electronics" }
